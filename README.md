# ABAP Learning 

> In this repository, you will find exercises for the ABAP learning path.

## Prerequisites

- VS Code or Eclipse with a ABAP Server

### VS Code
- [Standalone ABAP Development Extension Pack]
- node js
- clone this repository
- pnpm 
    ```bash
    npm install -g pnpm
    ```

### Eclipse
- ADT (ABAP Development Tools)
